# Watt Wise

This is the frontend for the Campus Energy Zen application, a platform for monitoring and managing campus energy consumption.

## Features

*   **Real-time Dashboard**: A comprehensive dashboard that displays live energy metrics, including campus load, solar generation, grid status, and battery level.
*   **Live Monitoring**: A detailed view of the campus energy infrastructure, with real-time graphs of energy flows and a status overview of all connected devices.
*   **AI Strategy Center**: An AI-powered tool for generating and managing energy-saving strategies. It includes a chatbot for interacting with the AI and asking questions about the energy data.
*   **Automated Reports**: A feature for generating and viewing energy performance reports.

## Getting Started

### Prerequisites

*   Node.js and npm (or bun)

### Installation

1.  Clone the repository:
    ```sh
    git clone <repository-url>
    ```
2.  Navigate to the `campus-energy-zen` directory:
    ```sh
    cd campus-energy-zen
    ```
3.  Install the dependencies:
    ```sh
    npm install
    # or
    bun install
    ```

### Running the Application

To start the development server, run:

```sh
npm run dev
# or
bun run dev
```

The application will be available at `http://localhost:8080`.
